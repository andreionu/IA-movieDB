Pașii metodologiei
Pasul 1
Alegerea setului de date de pe Kaggle

The Movies Dataset (Rounak Banik, Kaggle)

  • movies_metadata.csv — 45,466 filme cu genuri, overview, poster_path, vote_average
  • credits.csv — actori și echipă (JSON), din care extragem directorul
  • keywords.csv — cuvinte cheie per film (JSON)
  • ratings_small.csv — ~100k rating-uri reale (userId, movieId, rating)
  • links_small.csv — mapare movieId (MovieLens) ↔ tmdbId
Pasul 2
Pregătirea datelor
  • Parsare coloane JSON: genres, cast (top 3 actori), crew → extrage Director, keywords
  • Join: movies_metadata ← credits ← keywords pe id → 45,420 filme
  • Mapare rating-uri: ratings_small → links_small → tmdbId → titlu film → {{ stats.n_ratings }} rating-uri valide
  • Normalizare user-mean: scădem media fiecărui utilizator pentru a reduce bias-ul
  • Eliminare duplicate, filme adulte, titluri lipsă
Pasul 3
Alegerea algoritmilor de Machine Learning

Au fost implementați doi algoritmi complementari:

  • SVD TruncatedSVD (Collaborative Filtering) — descompune matricea utilizator×film în {{ stats.n_components }} vectori latenți care capturează preferințe implicite. Filme similare = filme care atrag același tip de utilizatori.
  • TF-IDF TF-IDF + Cosine Similarity (Content-Based) — construiește un vector TF-IDF din genuri, cuvinte cheie, regizor și actori. Filme similare = filme cu conținut textual similar.
Pasul 4
Antrenarea modelelor

SVD:

  • Matrice user-item: {{ stats.n_users }} utilizatori × {{ stats.n_films }} filme
  • TruncatedSVD(n_components={{ stats.n_components }}) pe matricea normalizată → varianță explicată: {{ stats.var_explained }}%
  • Similaritate item-item precomputată (cosine similarity în spațiu latent) pentru toți cei {{ stats.n_films }} filme

TF-IDF:

  • make_soup: genuri + cuvinte cheie + regizor (×2 pentru pondere) + actori top-3
  • TfidfVectorizer(max_features=10,000, ngram_range=(1,2))
  • Similaritate cosine precomputată pentru top-50 recomandări per film
Pasul 5
Validarea modelelor

Split 80% antrenare / 20% testare pe rating-urile mapate:

  • RMSE SVD: {{ stats.rmse }} vs. Baseline: {{ stats.rmse_baseline }} → îmbunătățire +{{ stats.improvement }}%
  • MAE model SVD: {{ stats.mae }}
  • Jaccard Similarity (coerența recomandărilor pe genuri): SVD = {{ stats.jaccard_svd }}, TF-IDF = {{ stats.jaccard_cb }}
Pasul 6
Utilizarea modelelor pentru predicții
  • get_similar_movies(title, n) — top-N filme similare via SVD item-item
  • get_cb_recommendations(title, n) — top-N filme similare via TF-IDF content-based
  • get_user_recommendations(user_id, n) — predicții per utilizator (SVD reconstructed matrix)
  • Postere: TMDB API v3 → descărcare locală → cache în static/posters/
